\doxysubsection{Detailed Description}
\doxyref{Composite\+Filter}{p.}{class_composite_filter} aggregates multiple log filters. 

This class implements the \doxyref{ILog\+Filter}{p.}{class_i_log_filter} interface by combining several filters. A log line passes the composite filter if and only if it satisfies all added filters.
